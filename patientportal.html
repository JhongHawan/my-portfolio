Patient Portal Database

SQL relational database built for a patient portal application.

Previous Next

SUMMARY

I built this database for a class assignment where we learned about database design and development. We specifically focused on learning sql relational databases and this assignment showed me all the different elements of database design.

The assignment is a database designed to capture information about patients and their appointment information. It includes multiple tables, stored procedures, and testing code.

Technologies Used

These are the coding technologies I used in this project.

  • SQL
  • Entity-relationship modeling
  • Relational model
  • SQL Server Management Studio (SSMS)
